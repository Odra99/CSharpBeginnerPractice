# C# Beginner Practice

- **Course Author**: [Mosh Hamedani](https://github.com/mosh-hamedani)
- **Course**: [C# Basics for Beginners: Learn C# Fundamentals by Coding](https://www.udemy.com/course/csharp-tutorial-for-beginners/)

A comprehensive collection of C# exercises designed for beginners to learn and practice fundamental programming concepts. This project covers control flow, arrays, lists, strings, and file handling.

## 📋 Project Overview

This project provides interactive command-line exercises that help beginners understand core C# programming concepts through practical implementation. Each exercise is self-contained and includes user interaction and validation.

## 🚀 Features

The project includes **17 interactive exercises** across 4 categories:

### 1. **Control Flow Exercises** (5 exercises)
- Count numbers divisible by 3 between 1 and 100
- Sum user-entered numbers until 'ok' is entered
- Calculate the factorial of a number
- Guess the number game (1-10 with limited attempts)
- Find the maximum number in a list

### 2. **Array and List Exercises** (5 exercises)
- Facebook likes counter (array manipulation)
- Reverse a name using character arrays
- Sort 5 unique numbers
- Select unique numbers from a list with duplicates
- Find the three smallest numbers in a list

### 3. **String Exercises** (5 exercises)
- Validate consecutive/non-consecutive numbers in a string
- Check for duplicates in a string
- Validate time format in 24-hour format
- Convert strings to PascalCase
- Count vowels in a string

### 4. **File Exercises** (2 exercises)
- Read a file and count the total number of words
- Find the longest word in a file

## 🛠️ Requirements

- **.NET 8.0** or higher
- **C# 12.0** or compatible version
- Visual Studio 2022 or Visual Studio Code with C# extension
